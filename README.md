clj-workshop-examples                                                                                                                                 
=====================

Compojure
---------

    lein ring server
    
Kafka
-----

    lein run-main-kafka

Hazelcast
---------

For hazelcast do this first:

    git clone https://github.com/runa-labs/clj-hazelcast
    cd clj-hazelcast
    lein install

Then in the examples directory
    
    lein run-main-hazelcast
